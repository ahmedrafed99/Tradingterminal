Transitions
background 0.15s color 0.15s opacity 0.15s border-color 0.15s transform 0.15s
Standard Transition Duration
0.15s (150ms)
Every interactive state change (hover, focus, active, open/close) must be animated at 150ms. No exceptions. Popovers use 180ms ease-out for entrance animation.
disabled: opacity 0.5
Disabled State
opacity: 0.5
Always 0.5. Never 0.3, 0.4, or any other value. cursor: not-allowed or default.
